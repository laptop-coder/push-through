## Tech Stack

- **Android/iOS/Linux/Windows/macOS app**: Flutter (Dart)

## Getting Started

You can download builded app from the [Releases page](https://github.com/laptop-coder/push-through/releases).

Run app in the development mode:

```bash
# Clone the repository
git clone https://github.com/laptop-coder/push-through.git
cd ./push-through/app
# Run
flutter run
```

You can also specify OS:

```bash
flutter run -d Linux
```
